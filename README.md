# Quik Pods

## Getting Started

First install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Testing

This project uses [Vitest](https://vitest.dev/) to run the tests. To run the tests, use the following command:

```bash
npm run test
# or
yarn test
# or
pnpm test
# or
bun test
```
